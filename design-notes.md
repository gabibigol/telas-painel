# TikTok Shop - Design Notes

## Página de Produto (PDP)

### Estrutura Visual
- Header com logo da loja, ícone de carrinho e menu
- Galeria de imagens do produto (swiper/carousel)
- Badges: "Hot Deal" (laranja), "New customer deal" (verde)
- Nome do produto
- Preço em destaque (rosa/vermelho TikTok)
- Rating com estrelas + número de vendas
- Seletor de cor (thumbnails circulares)
- Seletor de tamanho (botões: S, M, L, XL)
- Botões de ação: "Add to cart" (outline) + "Buy now" (rosa sólido)
- "Buyer Protection Program" no rodapé

### Cores
- Rosa TikTok: #FE2C55
- Verde deal: #00C853
- Laranja hot deal: #FF6D00
- Fundo: #FFFFFF
- Texto: #161823

## Checkout

### Estrutura Visual
- Header com seta voltar + "Checkout"
- Endereço de entrega (editável)
- Shipping method com logo da loja
- Item do pedido (imagem, nome, variante, preço, quantidade)
- Standard shipping: Free + data estimada
- Payment method (Card, PayPal, Apple Pay)
- Order summary (Total)
- Botão "Place order" (rosa sólido)
- "Buyer Protection Program" no rodapé

### Elementos UI
- Radio buttons para métodos de pagamento
- Ícones de cartão de crédito
- Separadores sutis entre seções
